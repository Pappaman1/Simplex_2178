THE PROGRAM READS IN THE MAP.TXT FILE AND CONSTRUCTS THE MAZE FROM THERE!
THE MAZE IS CURRENTLY HARDED CODED TO BE 5x5. 
When you build the program, it copies the map.txt into the _Binary folder and you can edit it from there.
0 are the walls
Anything else is the weight of the path.
98 is the goal value
99 is the start value
